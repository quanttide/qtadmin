# qtadmin_studio

量潮管理后台客户端。

## 目录

```
lib/
├── blocs/        # BLoC 状态管理
├── models/       # freezed 数据模型
├── sources/      # 数据源抽象（base + file_source + bundle_source）
├── screens/      # 页面
├── views/        # 组件
├── theme.dart    # 颜色工具
├── constants.dart# UI 映射常量
├── main.dart
└── router.dart   # RouteConfig + AppRouter
```
