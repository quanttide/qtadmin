## 1. 功能概述

从创始人工作日志中提取结构化事件记忆，并生成清洗后的日记。

**处理流程**:
1. 按日期分组 `raw/` 目录下的原始 markdown 文件
2. 读取同一天的所有原始内容并合并
3. 调用 LLM 提取事件 → 保存到 `{date}/meta.jsonl` (JSONL)
4. 调用 LLM 清洗日志 → 保存到 `{date}/diary.md` (MYST)
